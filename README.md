# Personal Site v1

A minimal static personal website for **Runxin Shao**.

## Files
- `index.html` — page content
- `styles.css` — site styling

## What to edit next
You should replace or refine:
- the **About** paragraph
- the **Selected Projects** placeholders
- the **Experience** section
- the **Coursework / Technical Direction** list

## Preview locally
Open `index.html` directly in a browser.

## Deploy with GitHub Pages
1. Create a GitHub repository (for example `runxinshao.github.io` or `personal-site`).
2. Upload `index.html` and `styles.css`.
3. If using GitHub Pages, enable Pages in the repository settings.

## Deploy with Vercel
1. Push this folder to a GitHub repo.
2. Import the repo into Vercel.
3. Deploy as a static site.

## Notes
This version is intentionally simple: clean, easy to edit, and suitable as an academic + engineering homepage.
